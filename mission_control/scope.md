# Project Scope

## App Summary
Habit-bit is a premium personal habit tracker that transforms routine consistency into a visual journey. It uses "The Living Bridge" visualization to represent streaks, making every completed habit feel like building a path toward a better self.

## User Persona
- **Who**: Productivity enthusiasts and "visual thinkers" who find text-based trackers boring.
- **Goal**: Maintain long-term habits through rewarding, high-end visual feedback.
- **Pain**: Losing motivation due to clinical, uninspiring interfaces that don't celebrate progress.

## Feature Priority
| Priority | Feature | Notes |
|---|---|---|
| P0 (MVP) | **Bento Dashboard** | Premium dark mode layout for habit overview. |
| P0 (MVP) | **Habit CRUD** | Create, edit, and categorize habits with icons/colors. |
| P0 (MVP) | **The Living Bridge** | Basic streak visualization (bridges/flames) using Framer Motion. |
| P1 (Quality) | **Tasks** | High-fidelity task list with bento integration. |
| P1 (Quality) | **Calendar** | Visual schedule and ritual planning. |
| P1 (Quality) | **Books** | Reading habit tracking and library management. |
| P1 (Quality) | **Map** | Location-based triggers and journey visualization. |
| P1 (Quality) | **Quote Engine** | Dynamic motivational quotes on habit interaction. |
| P1 (Quality) | **PWA Support** | Installable mobile experience with offline tracking. |
| P2 (Future) | **Advanced Analytics** | Deep dives into habit performance over months/years. |
| P2 (Future) | **Momentum Score** | Aggregated daily "Heat" score. |

## Out of Scope
- Native iOS/Android mobile applications.
- Social features (leaderboards, friend follows).
- Complex health data integrations (manual tracking only).
