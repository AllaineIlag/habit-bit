# ⚠️ shallow — Clarity Requests

This folder contains tasks that Swain flagged as **too vague to safely delegate**.

When you see a `.shallow` file appear here, it means Swain needs more information before he can dispatch the task to a regional team.

---

## What to Do

1. Open the `.shallow` file — Swain will have written specific questions
2. Answer the questions
3. Drop a **new** `.raw_task` in `/raw_tasks/` with the full, clarified request
4. The original task is now in `/tasks_log/pending_clarity/` for reference

---

## Why This Exists

An agent that guesses what you want and gets it wrong wastes:
- Your OpenClaw credits
- Your API rate limits
- Your time reviewing broken output

The shallow gate exists to prevent that.
